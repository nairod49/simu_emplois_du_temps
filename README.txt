Projet de la facultés des sciences d'Angers 2020-2021

Projet sur la simulation d'emplois du temps

Réalisé par Dorian Letort

Pour instalé se projet placé le dossier dans votre local host 
Puis aller dans votre localhost::phpMyAdmin et crée une base 

Ensuite aller dans le fichier BDD.php pour modifier l'user, le mot de passe et votre nom de table 
Faite de même a la fin du fichier interface.php

Et enfin aller sur localhost/simu/interface.php

Attention lors du chargement des données de la maquette les bouton radio se réinitialise pensé a recocher votre précedent choix.
